Classes and Objects
What is Class?
- A blueprint in C# that defines variables and methods for an object.
What is Objects?
- An instance of a class representing its concrete form in a program.
Value and Reference Types
A. Value and Reference Types
- Stores data in the stack, used for static memory allocation with consistent memory requirements.
B. Reference Types
- Store data in the heap, used for dynamic memory allocation for objects; the heap’s address is stored in the stack.
Encapsulation
- Hides internal workings of a class, combining members inside. It restricts external access and provides benefits
like data security, flexibility, and code isolation.

Access Modifiers
A. Public
- Accessible outside the class.
B. Private
- Only accessible within the class.
C. Protected
- Accessible by the same class or derived classes.
Constructors
- Special methods automatically executed when an object is created. They have the same name as the class,
no return type, and can have parameters.
Properties
-Provides a flexible mechanism to read, write, and compute values of private fields in a class.
They can use get and set accessors and can omit them to control behavior.

Auto-Implemented Properties: Allow for quick and simple declarations of private members with minimal code
