What is C#?
A. It is an elegant object-oriented language that enables users to build secure,
robust applications that can run on .NETFramework.

Parts of .NETFramework
• The Common Language Runtime (CLR) is the foundation of . NETFramework
• it also has a class library in which it is a collection of classes, interfaces and
value types that allows you to accomplish a wide variety of tasks
B. C# uses . NETFramework extensively to do common tasks and provide different functions
Variables
A These are also known as data types
B. They define what kind of data can be stored, memory needed for the task, and provide different functions
Supported data types in C#
1. Int: whole numbers
2. String: Texts
3. Float: small decimal numbers
4. Double: accommodates bigger decimal numbers than float
5. Char: supports single alphanumerical characters
6. Bool: only accepts boolean values; true or false
Printing Texts
1. Uses Console.ReadLine()
2. Its input is automatically assigned as string
3. Input values can be changed by using;
A. Convert.ToDouble(), converts the assigned string as double
B. Convert.ToBoolean(), converts the assigned string as a boolean
C. Convert.ToInt(), converts the assigned string as int based on
the specific size bit of the integer Int 16, int 32 and int 64

Comments
A. These are explanatory statements
B. The compiler ignores the comment
TWO TYPES OF COMMENTS:
1. Single line comment
- Begins with two forward slashes (//)
2. Multi line comments
- Has multiple lines
- Begins with /* and ends with */
The var keytwords
• allows the compiler to automatically determine a variables type based on the assigned value
• must be initialized to avoid errors
Contants
• These are the types of values that cannot be changed once assigned a value.
Arithmetic Operations
• Commonly used to perform mathematical operations on values
• Most operators are either arithmetic or logical
A. Supported operators and its symbol - (+) addition
- (-)subtraction
- (*)multiplication
- (/) division
- (%) modulus
Assignments and Increments
• assigns a value from left to right side of the operator
• denoted by the symbol (=)
•C# also provides Compound Assignments Operators, that performs an operation
and assignment at the same time, this can be done by adding the appropriate
arithmetic operator sequence or precedence before the equal sign
A. Increment denoted by (++)
- Increases the value of int by 1
Types of increment - Prefix: increments the value first before evaluating the expression
- Postfix: evaluates the expression before incrementing
B. Decrement denoted by (--)
- Decreases the value of int by 1
Types of decrement
- Prefix: decrements the value first before evaluating the expression
- Postfix: evaluates the expression before decrementing
